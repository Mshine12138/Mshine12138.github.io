    Markdown 编辑器初体验

    推荐来源：

    也许是B站最好的 Markdown 科普教程 | 15 款顶级笔记软件测评推荐-哔哩哔哩

    写作软件综合测评

    Simplenote

    预览

    优点

    1. 免费
    2. 非常简单简约，干净无广，视觉舒适不生硬。
    3. 打开即用，没有学习成本。
    4. 在线同步还可以，虽然有延迟，但是网络ok的话几秒钟的事。
    5. 不需要vpn。

    缺点

    1. 没有双视图。硬要试试的话，可以开两个窗口或两个设备，但是响应速度比较慢，有延迟，非常不方便。
    2. 导出只能一次性全导出，不能单个笔记导出。
    3. 在编辑列表的时候，回车后没有自动生成列表符。
    4. ios端没有渲染视图。
    5. 导出不是md文件，有tag的话，txt文件里面包含，需要手动c/p。
    6. 不支持内嵌html，尽管会忽视<>的标记。
    7. 分享的预览和编辑中的预览不一样

    Gingko Writer

    预览

    这一款很牛逼。预设里可以支持论文结构、写作框架、头脑风暴，而我觉得这远远不止。这种全视图的树状信息结构结合markdown的写作流，潜力绝对远不止如此。

    优点

    1. 免费
    2. 支持html语法，可以改变文本颜色和布局
    3. 支持多人协作
    4. 在线同步非常快
    5. 不需要vpn

    缺点

    1. 创建列表没有回车自动列表符
    2. 略有一些小操作会意想不到，以致于需要习惯，学习成本不低
    3. 只有网页端，没有客户端
    4. 只能在线
    5. 导出不是md文件，但是其他形式还挺多的。

    Obsidian

    预览

    优点

    1. 免费
    2. 简约
    3. 系统内保存为md格式
    4. 可导入其他平台的数据
    5. 可探索的功能强大
    6. 支持所见即所得
    7. 支持内嵌html
    8. 不需要vpn

    缺点

    1. 同步需要收费
    2. 学习成本略高，需要花时间摸索
    3. 外链需要本地软件去打开

    总结

    • Simplenote 论快速不如记事本，论便捷不如备忘录，论同步传输不如微信助手。
    • Gingko Writer 在特定的树状结构情景下值得一用。
    • Obsidian 非常适合统筹全部的内容，可以作为主要工具使用。
